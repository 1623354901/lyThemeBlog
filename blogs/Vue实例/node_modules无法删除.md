1、[安装 npm](https://so.csdn.net/so/search?q=安装npm&spm=1001.2101.3001.7020)包–**rimraf**，很小的一个工具包

```
npm install rimraf -g
```

2、在**cmd**指令下，进入所需删除的 node_modules 文件夹的**位置**（路径）

或 在 VScode 等开发工具的终端进入到对应**位置**（路径）

3、输入以下指令进行删除

```
rimraf node_modules
```
