> > > 最常用的两个：

#### 字符串转化为数组：

```js
let str = 'helloworld'
str.split('')
//结果：['h','e','l','l','o','w','o','r','l','d']
```

#### 数组转为字符串的：

```js
var a = [1, 2, 3, 4, 5]
var s = a.join('-')
//s:'1-2-3-4-5'
```
